# 电子时钟

## 使用说明
- 在布局中添加
```
<zd.com.clock.ElectronicClock
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:textColor="@android:color/black"
        android:textSize="14sp"
        app:textFormatStr="HH:mm:ss" />
```
textFormatStr显示的格式

- 运行即可显示
